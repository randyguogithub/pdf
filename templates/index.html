Developer FAQs

1. General information and availability

1.1 What is ReportLab?

ReportLab Inc. is a software company developing open-source and commercial products for generating PDF files. It is a collection of open source Python language modules organized and supported by ReportLab. The modules are primarily concerned with automating methods for generating Portable Document Format (PDF) files, but there is also a cross-platform vector graphics library able to generate bitmap and vector charts and drawings.

1.2 How do I get the ReportLab Toolkit?

If you have not already done so, you will need to install the Python language from www.python.org. If you want to include bitmap images, install the Python Imaging Library. Thereafter, you need the latest stable version of the ReportLab package from our downloads page.
